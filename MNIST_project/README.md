## Project Overview

This project implements a Convolutional Neural Network (CNN) for handwritten digit classification using the MNIST dataset.

Handwritten digit classification is a fundamental Computer Vision task. In this project, a CNN model is used to classify digits from 0 to 9 with high accuracy.

CNN is chosen because it preserves the spatial structure of images and performs better than traditional fully connected neural networks.

## Model Performance

Accuracy Achieved: 90.04%

## Dataset

The project uses the MNIST dataset which contains:

- 60,000 training images
- 10,000 testing images
- 28x28 grayscale images
- Digits from 0–9
